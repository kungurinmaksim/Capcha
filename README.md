# 🧩 Капча — Пазл с выбором имени

**Игра-головоломка на Pygame с системой состояний, меню и выбором имени игрока.**

## 🎮 Об игре

Это расширенная версия классического пазла, где игрок должен собрать разрезанную на фрагменты картинку. В отличие от простого пазла, проект реализован с использованием паттерна проектирования **«Состояние» (State)**, что позволяет легко переключаться между экранами: заставка → меню → ввод имени → игра.

### 🌟 Особенности

- 🎭 **Панель заставки** — приветствие "Не ты ли робот?" с мигающей подсказкой
- 📋 **Главное меню** — навигация с подсветкой выбранного пункта
- ✏️ **Ввод имени игрока** — до 10 символов (буквы и цифры)
- 🧩 **Классический пазл 4×4** — 16 фрагментов, собираемых в правильном порядке
- 🔢 **Счётчик перестановок** — отслеживает количество ходов
- 🖼️ **Случайные картинки** — новая картинка при каждом запуске игры
- 🔄 **Возврат в меню** — по нажатию пробела во время игры
- 🏆 **Отображение имени** — имя игрока показывается на экране игры
- 🎨 **Цветовая схема** — использование модуля `colors.py` для централизованного управления цветами

## 🕹️ Управление

### Заставка (SplashScreen)
| Действие | Управление |
|----------|------------|
| Продолжить | Нажатие любой клавиши или клик мыши |

### Главное меню (MenuScreen)
| Действие | Управление |
|----------|------------|
| Перемещение вверх | Стрелка ВВЕРХ |
| Перемещение вниз | Стрелка ВНИЗ |
| Выбор пункта | ПРОБЕЛ или ENTER |

### Пункты меню
| Пункт | Действие |
|-------|----------|
| играть | Начать сборку пазла |
| Выбрать имя игрока | Перейти к экрану ввода имени |
| Выйти | Закрыть игру |

### Экран ввода имени (NameScreen)
| Действие | Управление |
|----------|------------|
| Ввод символов | Буквы (A-Z, a-z) и цифры (0-9) |
| Удаление символа | BACKSPACE |
| Подтверждение имени | ENTER |

### Игровой экран (GameScreen)
| Действие | Управление |
|----------|------------|
| Выбор фрагмента | Клик левой кнопкой мыши |
| Обмен фрагментов | Выбрать первый → клик по второму |
| Отмена выбора | Клик по выбранному фрагменту |
| Возврат в меню | ПРОБЕЛ |
| Выход из игры | Закрыть окно |

## 📸 Как играть

1. **Запустите игру** — появится заставка "Не ты ли робот?"
2. **Нажмите любую клавишу** для перехода в главное меню
3. **Выберите имя игрока** (опционально):
   - Перейдите в пункт "Выбрать имя игрока"
   - Введите имя (до 10 символов)
   - Нажмите Enter для сохранения
4. **Выберите "играть"** в главном меню
5. **Собирайте пазл**:
   - Кликайте по фрагментам для их выбора
   - Выбранный фрагмент подсвечивается зелёным
   - Меняйте фрагменты местами, чтобы восстановить картинку
6. **Отслеживайте прогресс**:
   - Счётчик перестановок показывает количество ходов
   - Имя игрока отображается в правом верхнем углу
7. **При победе** появится поздравление
8. **Нажмите ПРОБЕЛ** в любой момент, чтобы вернуться в меню

## 🏗️ Архитектура проекта (Паттерн State)

Проект построен на паттерне проектирования **«Состояние»**, что позволяет легко управлять переключением между экранами:

```python
class State(abc.ABC):
    - handle_events(events)  # Обработка ввода
    - update()               # Обновление логики
    - draw(screen)           # Отрисовка
```

### Доступные состояния:

| Класс | Назначение |
|-------|------------|
| `SplashScreen` | Заставка с мигающей подсказкой |
| `MenuScreen` | Главное меню (3 пункта) |
| `NameScreen` | Экран ввода имени игрока |
| `GameScreen` | Основная игра (пазл) |

### Переходы между состояниями:

```
SplashScreen → MenuScreen
MenuScreen → GameScreen
MenuScreen → NameScreen
MenuScreen → exit (закрытие)
NameScreen → MenuScreen (после ввода имени)
GameScreen → MenuScreen (по пробелу)
```

## 🔧 Технические детали

### Параметры игры

| Параметр | Значение |
|----------|----------|
| Размер окна | 800×800 пикселей |
| Сетка пазла | 4×4 (16 фрагментов) |
| Отступ между фрагментами | 2 пикселя |
| FPS | 60 кадров/сек |
| Фон | Зелёный (из colors.GREEN) |
| Шрифт | Системный, размер 64 |

### Механика пазла

1. **Загрузка изображения** — случайный выбор из папки `pictures`
2. **Нарезка** — деление на ROWS × COLS (4×4)
3. **Перемешивание** — случайный порядок фрагментов
4. **Обмен** — два клика по разным фрагментам
5. **Подсветка** — зелёная рамка вокруг выбранного фрагмента
6. **Победа** — сравнение текущего порядка с оригиналом

### Система имён

- Имя по умолчанию: `"АНОНИМ"`
- Максимальная длина: 10 символов
- Допустимые символы: буквы (A-Z, a-z) и цифры (0-9)
- Сохраняется в глобальной переменной `player_name`
- Отображается на экране игры в правом верхнем углу

## 🐛 Возможные проблемы и решения

### Ошибка: `ModuleNotFoundError: No module named 'colors'`

**Решение**: Создайте файл `colors.py` в той же папке:

```python
# colors.py
GREEN = (0, 255, 0)
BLACK = (0, 0, 0)
WHITE = (255, 255, 255)
```

### Ошибка: `pygame.error: Couldn't open ...`

**Причина**: Отсутствует папка `pictures` или в ней нет изображений

**Решение**:
```bash
mkdir pictures
# Добавьте хотя бы одно изображение в папку pictures
```

### Ошибка: `pygame.font not initialized`

**Решение**: Убедитесь, что `pygame.font.init()` вызывается после `pygame.init()`

### Имя не сохраняется между сессиями

**Особенность**: Имя хранится только в памяти и сбрасывается при перезапуске игры

**Улучшение**: Можно добавить сохранение в файл
