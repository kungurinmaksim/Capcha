
# 🧩 Пазл — картинка-головоломка

**Игра-головоломка на Pygame**, где нужно собрать разрезанную картинку, меняя местами фрагменты.

## 🎮 Об игре

Игра загружает случайное изображение из папки `pictures`, разрезает его на 16 фрагментов (4×4) и перемешивает их. Ваша задача — переставлять фрагменты местами, чтобы восстановить исходное изображение. Счётчик перестановок отслеживает количество ваших ходов.

### 🌟 Особенности

- 🖼️ **Случайная картинка** — каждый раз новая картинка из папки
- 🧩 **Сетка 4×4** — классический формат пазла (16 фрагментов)
- 🖱️ **Простое управление** — клик для выбора фрагмента и клик для обмена
- 🔢 **Счётчик ходов** — показывает количество перестановок
- ✅ **Определение победы** — автоматическое поздравление при сборке
- 🎨 **Визуальная индикация** — выбранный фрагмент подсвечивается зелёным

## 🕹️ Управление

| Действие | Управление |
|----------|------------|
| Выбор фрагмента | Клик левой кнопкой мыши по фрагменту |
| Обмен фрагментов | Выбрать первый фрагмент → кликнуть по второму |
| Отмена выбора | Кликнуть по уже выбранному фрагменту |
| Выход из игры | Закрыть окно |

## 📸 Как играть

1. **Запустите игру**
2. На экране появится перемешанный пазл из 16 фрагментов
3. **Кликните** по любому фрагменту — он подсветится зелёным
4. **Кликните** по другому фрагменту — они поменяются местами
5. Счётчик перестановок внизу экрана показывает количество ходов
6. Продолжайте менять фрагменты, пока не соберёте исходную картинку
7. При победе появится поздравительное сообщение

### Механика

1. **Загрузка изображения**:
   - Случайный выбор файла из папки `pictures`
   - Изображение загружается с помощью `pygame.image.load()`

2. **Нарезка на фрагменты**:
   - Изображение делится на сетку 4×4
   - Размер фрагмента = `ширина_изображения / 4` × `высота_изображения / 4`
   - Каждый фрагмент сохраняется как отдельная поверхность Pygame

3. **Перемешивание**:
   - Фрагменты перемешиваются случайным образом
   - Исходный порядок сохраняется для проверки победы

4. **Логика обмена**:
   - Клик по фрагменту выделяет его
   - Клик по другому фрагменту меняет их местами
   - Счётчик ходов увеличивается на 1

5. **Определение победы**:
   - Текущий порядок сравнивается с исходным
   - При совпадении выводится сообщение "Ура, картинка собрана!"

## 🎯 Игровая механика

### Визуальная индикация
- **Выбранный фрагмент** — подсвечивается зелёной рамкой
- **Счётчик перестановок** — отображается внизу экрана
- **Сообщение о победе** — появляется по центру экрана

### Пример игрового процесса

```
Начало: Фрагменты перемешаны
  ↓
Клик по фрагменту A → выделение
  ↓
Клик по фрагменту B → обмен A↔B, +1 ход
  ↓
Повторять, пока не соберётся исходное изображение
  ↓
Победа! Поздравление на экране
```

## ⚙️ Установка и запуск

### Требования
- Python 3.6 или выше
- Pygame (`pip install pygame`)

### Установка Pygame

```bash
pip install pygame
```

### Запуск игры

```bash

# 2. Создайте папку 'pictures' в той же директории

COLS = 6
```

## 🐛 Возможные проблемы и решения

### Ошибка: `pygame.error: Couldn't open ...`

**Причина**: Отсутствует папка `pictures` или в ней нет изображений

**Решение**:
```bash
mkdir pictures
# Добавьте хотя бы одно изображение в папку pictures
```

### Ошибка: `ModuleNotFoundError: No module named 'pygame'`

**Решение**:
```bash
pip install pygame
```

### Изображение не помещается в окно

**Решение**: Измените размер окна в коде или используйте изображения меньшего размера

### Тормозит игра

**Решение**: Уменьшите FPS в коде (например, на 30)
